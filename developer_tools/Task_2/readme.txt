Открыть todomvc\index.html в браузере.
Открыть консоль. В консоли увидите ошибку.
Задача. C помошью дебаггер инструментов разобраться почему происходить ошибка. Исправить её. Вместо ошибки информация должна логироваться в консоль.


Ответ:
Выбрала вариант, в котором убираю функцию log, файла app-view.js. Ошибка происходила из-за того, что log вызывал функцию console, которая была не определена. Чтобы вывести значения в консоли, я добавила в строку log('Completed: ' + completed, 'Remaining: ' + remaining); - window.console.log, тем самым вызвав функцию консоли браузера, а не определенную функцию  в файле. 
результат: 
window.console.log('Completed: ' + completed, 'Remaining: ' + remaining);

вывело в консоли: 
Completed: 0 Remaining: 0
Completed: 0 Remaining: 0
